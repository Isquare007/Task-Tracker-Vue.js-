# A Task Tracking App Built with Vue.js
# cd to the task_manager

## to start the backend:
### Install the json server
```
sudo npm i json-server
```
### Run the backend server
```
npm run backend
```

## To start the front end:
### To run dev server :
```
sudo npm run dev
```
### TO run production server:
```
sudo npm i -g serve
``` 
```
serve -s dist
```
